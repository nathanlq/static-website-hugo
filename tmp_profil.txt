PROJET DATAKHI #2 : HUN PREDICTIONS
-------------------------------------
Pipeline de machine learning pour la prevision d'affluence restaurant.
Prediction du nombre de couverts par jour/periode avec XGBoost quantile.
> 28 modeles independants (2 variantes x 2 periodes x 7 jours)
> Feature engineering avance : lags, rolling stats, meteo, calendrier,

> Loss function custom Huber, calibration des intervalles de confiance
> Collecte depuis Microsoft Fabric (Azure SQL), export CSV format francais
> Tests de non-regression (Pearson >= 0.999), workflow Makefile complet

construire un referentiel enrichi des entreprises actives.
> Client API avec rate limiting, pagination par curseur
> Telechargement bulk en Parquet (PyArrow) : 13M unites legales, 35M etab.
> Enrichissement API (dirigeants, GPS, NAF 2025)
> Jointure PyArrow entre unites legales et etablissements
> Conversion NAF 2025 depuis XLSX officiel INSEE
> Objectif : alimentation PostgreSQL avec sync incrementale hebdomadaire
> Stack : Python, PyArrow, Requests, PostgreSQL (cible)
